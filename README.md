# Gradelytics-
●	Developed a full-stack analytics platform to monitor and predict student academic performance using machine learning models. 
●	Built predictive models using Gradient Boosting algorithms to identify at-risk students based on attendance, assignments, and exam scores. 
●	Designed dashboards and KPI tracking features for analysing student progress, attendance trends, and academic insights. 
●	Implemented role-based access control for students, teachers, and administrators with secure authentication. 
●	Integrated REST APIs and MongoDB database for real-time student data management and analytics reporting. 
●	Enabled batch prediction and historical performance tracking through a Python Flask ML microservice.
